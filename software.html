Bash Tips & Tricks

  • how to copy data from one server to another (here from raijin to current directory):
    rsync -aruv mv7494@raijin.nci.org.au:/g/data/e14/mv7494/access-om2/archive/025deg_jra55_iaf/output0{50..57} .
  • check size of current folder:
    du -h --max-depth=1
  • calculating anomalies in CDO relative to 1979-2016 base period:
    cdo detrend -yseassub ifile.nc -yseasmean -selyear,1979/2016 ifile.nc ofile.nc &
  • saving aliases to save time typing folder directions all the time Open your favourite text editor; I use emacs:
    emacs ~/.bashrc
    alias emacs='emacs -nw' # opens the emacs text editor without a window
    alias data='/srv/ccrc/data15/z5180028' # alias which links to my data folder when I type 'data'
    alias ll='ls -ltrh' # get more and clearer info about files in folder
    # you can also load modules in bashrc to avoid typing it out everytime you open a new terminal
    module load ncview
    module load cdo
    module load netcdf-c/4.4.1.1

Type CTRL-X followed by CTRL-C to close the emacs text editor.

  • custom LaTeX command to highlight sections that still need to be addressed:
    \definecolor{myblue}{RGB}{49,65,149}
    \definecolor{lightorange}{RGB}{253,219,199}
    \newcommand{\note}{\textbf{\color{myblue}{\fontfamily{phv}\selectfont{\colorbox{lightorange}{\underline{Note:}} #1}}}}

To use the command, use \note{Text.} and the text in the bracket will be blue, bold and the Helvetica font.

  • LaTeX command to avoid double brackets when using citations:
\citep[see e.g.][]{meinen2000observations, lengaigne2012mechanisms}

gives as a result: (see e.g. Meinen and McPhaden 2000; Lengaigne et al. 2012)

  • restarting unix network printers so they show up in the list of available printers again
    sudo /etc/init.d/cups restart
